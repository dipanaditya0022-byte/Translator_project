## 🚀 Overview

Azure Translator Dashboard is a visually rich, cyberpunk-inspired web app built for real-time text translation. It combines a glowing neon interface, analytics widgets, quick language shortcuts, usage history, live metrics, and Azure-powered translation into one immersive dashboard.

This project is designed to look like a **next-gen sci-fi control panel** while still being practical and functional.

---

## ✨ Key Highlights

- ⚡ Real-time translation with Azure Translator

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone <your-repo-link>
cd translator_project
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the proxy server

```bash
node proxy-server.js
```

### 4. Open the dashboard
Open `index.html` in your browser or run with Live Server.

---

## 🔑 Azure Configuration

You need:
- Azure Translator API Key
- Azure Region

Open the dashboard and click the **Settings** button.
Then enter:
- API key
- Region name

Example:

```txt
Key: your_azure_translator_key
Region: eastasia
```
